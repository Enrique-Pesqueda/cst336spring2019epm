What is Ethical Hacking?

Ethical Hacking is also known as Penetration Testing which is used
to test if networks have any vulnerabilities in order to prevent loss
of data, or money. Ethical Hacking is specifically used only to allow
the security of a network or system to improve. Ethical Hackers generally
use the same tactics of a malicious hacker. When vulnerabilities are
found ethical hackers are required to report back.
